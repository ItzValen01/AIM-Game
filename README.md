# Proyecto: AIM Game

Es un pequeño proyecto de un juego de AIM.
Consiste en darle clic a un circulo lo mas rapido que puedas, con la dificultad de que el circulo en cada clic
cambia de lugar.

---

## 🚀 Tecnologías Usadas

### 📌 **Frontend**
- Astro
- ReactJS
- TailwindCSS

### 🖥️ **Backend**
No tiene backend.
